# Content Inputs

## Required

- Product name
- Short product description
- Target audience
- Core problem
- Key benefits
- Key features
- Technical specs

## Optional

- Customer segment
- Differentiators
- Use cases
- Proof points
- Pricing or packaging notes
- Compliance/security details
- Links to source material

## AI guidance when content is incomplete

- Ask for missing critical details when working interactively.
- Make reasonable placeholders only when explicitly allowed.
- Do not invent unsupported technical claims.
