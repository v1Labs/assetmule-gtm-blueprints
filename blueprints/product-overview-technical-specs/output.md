# Output Shape

## Recommended sections

- Title
- Subtitle
- Problem
- Solution
- Key benefits
- Key features
- Technical specs
- Optional proof / trust section
- Optional CTA or next step

## Acceptance criteria

- Reader can understand what the product does within 30 seconds.
- Benefits and features are distinct.
- Technical specs are structured and scannable.
- Claims are grounded in provided content.
- Output does not feel like generic AI marketing copy.
